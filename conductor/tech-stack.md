# Technology Stack

## 🧠 Digital Core & Logic
- **Bun:** High-performance JavaScript/TypeScript runtime for the primary CLI brain.
- **TypeScript:** Typed programming for robust, scalable system architecture.
- **React (Ink):** Building responsive, component-driven Terminal User Interfaces (TUI).
- **Python:** Powering the comprehensive system-level test suites (`pytest`).

## 🖥️ Web Dashboard (RAM)
- **Next.js:** Full-stack framework for the knowledge management interface.
- **React:** Component-based UI for the dashboard dashboard.
- **Tailwind CSS:** Utility-first styling for a sleek, responsive design.

## 🔌 Hardware Bridge & Firmware (MicroMax)
- **C++:** Low-level language for performant microcontroller firmware.
- **Arduino Framework:** Standardized ecosystem for hardware interaction.
- **PlatformIO:** Professional IDE and build system for embedded development.
- **Apex Serial Protocol (ASP):** Custom communication layer for AI-to-Hardware links.

## 🤖 AI & Agentic Stack
- **Anthropic / AWS Bedrock / Google Vertex:** Core LLM engines for high-level reasoning.
- **Model Context Protocol (MCP):** Universal standard for connecting agents to external tools and data.
- **OpenTelemetry:** Monitoring and observability across the agentic platform.

## 🛠️ Design & Build
- **KiCad:** Open-source EDA for professional-grade PCB layout and schematics.
- **Git:** Version control and distributed collaboration.
